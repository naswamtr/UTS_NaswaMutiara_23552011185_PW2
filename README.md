## 📌 About Project

Sistem Informasi Akademik adalah aplikasi berbasis web yang digunakan untuk mengelola berbagai data akademik seperti mahasiswa, jurusan, dan mata kuliah. Sistem ini dilengkapi dengan fitur autentikasi serta pengelolaan data menggunakan konsep CRUD (Create, Read, Update, Delete).

---

## 🚀 Features

Berikut adalah fitur utama yang tersedia dalam sistem:

### 🔐 Authentication
- **Register** → Pengguna dapat membuat akun baru  
- **Login** → Pengguna masuk ke dalam sistem  
- **Logout** → Keluar dari sistem dengan aman  

---

### 👤 Profile Management (CRUD)
Mengelola data profil pengguna:
- Create (Tambah data) > ada pada register
- Read (Lihat data)
- Update (Edit data)
- Delete (Hapus data)

---

### 🏫 Jurusan
Manajemen data jurusan:
- Tambah jurusan
- Lihat jurusan
- Edit jurusan
- Hapus jurusan

---

### 📚 Mata Kuliah
Manajemen data mata kuliah:
- Tambah mata kuliah
- Lihat mata kuliah
- Edit mata kuliah
- Hapus mata kuliah

---

### 🎓 Mahasiswa
Manajemen data mahasiswa:
- Tambah mahasiswa
- Lihat mahasiswa
- Edit mahasiswa
- Hapus mahasiswa
